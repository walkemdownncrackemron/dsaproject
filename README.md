# Data Structures & Algorithms Project
## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install pytest
   ```

3. Run the app:
   ```bash
   python -m src.ui.app
   ```

4. Run tests:
   ```bash
   pytest
   ```
